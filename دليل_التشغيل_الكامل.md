## 🛠️ تثبيت البرامج المطلوبة

### الخطوة 1: تثبيت VS Code

1. افتح المتصفح واذهب إلى: https://code.visualstudio.com/
2. اضغط على زر **Download** (تحميل)
3. اختر نظام التشغيل الخاص بك (Windows / Mac / Linux)
4. بعد التحميل، افتح الملف واتبع تعليمات التثبيت
5. اضغط **Next** (التالي) حتى تنتهي التثبيت
6. افتح VS Code

### الخطوة 2: تثبيت Node.js

1. افتح المتصفح واذهب إلى: https://nodejs.org/
2. اضغط على زر **LTS** (الإصدار المستقر) - هذا هو الزر الأخضر الكبير
3. سيتم تحميل ملف التثبيت
4. افتح الملف المحمل واتبع التعليمات:
   - اضغط **Next** (التالي)
   - اضغط **Next** مرة أخرى
   - اضغط **Install** (تثبيت)
   - انتظر حتى ينتهي التثبيت
   - اضغط **Finish** (إنهاء)

**للتأكد من التثبيت:**
1. افتح **Terminal** (الطرفية) في VS Code:
   - في VS Code: اضغط `Ctrl + ~` (الزر مع الزر فوق Tab)
   - أو من القائمة: **Terminal** → **New Terminal**
2. اكتب الأمر التالي واضغط Enter:
   ```
   node --version
   ```
3. إذا ظهر رقم مثل `v18.x.x` أو `v20.x.x`، فهذا يعني أن التثبيت ناجح ✅

### الخطوة 3: تثبيت PostgreSQL

#### على Windows:

1. افتح المتصفح واذهب إلى: https://www.postgresql.org/download/windows/
2. اضغط على زر **Download** للإصدار الأحدث
3. اختر **Windows x86-64**
4. سيتم تحميل ملف التثبيت
5. افتح الملف واتبع التعليمات:
   - اضغط **Next**
   - اختر المجلد الافتراضي واضغط **Next**
   - اختر المكونات الافتراضية واضغط **Next**
   - اضبط كلمة مرور: **postgres** (اكتبها واحفظها!)
   - اضغط **Next**
   - اختر المنفذ الافتراضي **5432** واضغط **Next**
   - اختر **Locale: English, United States** واضغط **Next**
   - اضغط **Next** ثم **Install**
   - انتظر حتى ينتهي التثبيت
   - اضغط **Finish**

#### على Mac:

1. افتح Terminal واكتب:
   ```
   brew install postgresql
   ```
2. ابدأ الخدمة:
   ```
   brew services start postgresql
   ```

#### على Linux (Ubuntu/Debian):

1. افتح Terminal واكتب:
   ```
   sudo apt-get update
   sudo apt-get install postgresql postgresql-contrib
   ```
2. ابدأ الخدمة:
   ```
   sudo service postgresql start
   ```

### الخطوة 4: إنشاء قاعدة البيانات

1. افتح **Terminal** في VS Code
2. اكتب الأمر التالي للدخول إلى PostgreSQL:
   ```
   psql -U postgres
   ```
3. سيطلب كلمة المرور: اكتب `postgres` واضغط Enter
4. اكتب الأوامر التالية واحدة تلو الأخرى:

   ```sql
   CREATE DATABASE noble_drip;
   ```

   اضغط Enter

   ```sql
   \q
   ```

   اضغط Enter (للخروج)

**ملاحظة**: إذا لم يعمل الأمر `psql`، جرب:
- على Windows: افتح **SQL Shell (psql)** من قائمة Start
- على Mac/Linux: استخدم `sudo -u postgres psql`

## 📂 فتح المشروع في VS Code

### الخطوة 1: الحصول على ملفات المشروع

إذا كان المشروع موجوداً بالفعل على جهازك:

1. افتح VS Code
2. من القائمة العلوية، اختر **File** → **Open Folder...**
3. اذهب إلى المجلد الذي يحتوي على المشروع (`noble-drip`)
4. اضغط **Select Folder** (اختيار المجلد)

### الخطوة 2: تثبيت التبعيات

1. تأكد أن **Terminal** مفتوح في VS Code
2. تأكد أنك في مجلد المشروع (يجب أن يظهر `noble-drip` في بداية السطر)
3. اكتب الأمر التالي واضغط Enter:
   ```
   npm install
   ```

**ما يحدث الآن؟**
- سيقوم npm بتحميل جميع المكتبات المطلوبة
- قد يستغرق هذا بضع دقائق
- انتظر حتى ينتهي تماماً
- سترى الكثير من الأسطر تظهر، هذا طبيعي ✅

**علامة النجاح:**
- عندما ترى سطر مثل: `added 360 packages`
- أو عندما يعود المؤشر إلى السطر الجديد بدون أخطاء

## 🗄️ إعداد قاعدة البيانات

### الخطوة 1: تحديث ملف .env

1. في VS Code، افتح الملف `.env` (تجده في قائمة الملفات على اليسار)
2. تأكد أن السطر التالي موجود:
   ```
   DATABASE_URL="postgresql://postgres:postgres@localhost:5432/noble_drip?schema=public"
   ```
3. إذا كان مختلفاً، عدله ليطابق هذا السطر
4. احفظ الملف: `Ctrl + S`

**شرح السطر:**
- `postgres` (الأول): اسم المستخدم
- `postgres` (الثاني): كلمة المرور
- `localhost`: عنوان الخادم (جهازك)
- `5432`: رقم المنفذ
- `noble_drip`: اسم قاعدة البيانات

### الخطوة 2: دفع schema إلى قاعدة البيانات

1. في Terminal، اكتب:
   ```
   npm run db:push
   ```

**ما يحدث الآن؟**
- سيقوم Prisma بإنشاء جميع الجداول في قاعدة البيانات
- سيستغرق بضع ثوانٍ

**علامة النجاح:**
- سترى رسالة: `✨ Done in X.XXs`
- أو رسالة: `Your database is now in sync with your Prisma schema`

### الخطوة 3: تشغيل البيانات الأولية (Seed)

1. في Terminal، اكتب:
   ```
   npm run db:seed
   ```

**ما يحدث الآن؟**
- سيقوم البرنامج بإنشاء:
  - حساب المسؤول (Super Admin)
  - الفئات الأولية للقهوة
  - إعدادات الموقع
  - أسعار بورصة القهوة
  - كوبون تجريبي

**علامة النجاح:**
- سترى رسائل مثل:
  ```
  🌱 Starting seed...
  ✅ Super Admin created: yuoseef01102@gmail.com
  ✅ Categories created
  ✅ Site Settings created
  ✅ Coffee Exchange Prices created
  ✅ Sample Coupon created
  🎉 Seed completed successfully!
  ```

### الخطوة 4: توليد Prisma Client

1. في Terminal، اكتب:
   ```
   npm run db:generate
   ```

**علامة النجاح:**
- سترى رسالة: `Prisma Client generated`

## 🔍 استكشاف الأخطاء

### مشكلة: "command not found: npm"

**الحل:**
- تأكد من تثبيت Node.js بشكل صحيح
- أعد تشغيل VS Code
- أعد فتح Terminal

### مشكلة: "connection refused" أو "could not connect to server"

**الحل:**
1. تأكد أن PostgreSQL يعمل:
   - على Windows: افتح **Services** وابحث عن **postgresql-x64-xx**
   - على Mac/Linux: `sudo service postgresql status`
2. إذا لم يكن يعمل، ابدأه:
   - على Windows: اضغط **Start** في Services
   - على Mac/Linux: `sudo service postgresql start`

### مشكلة: "password authentication failed"

**الحل:**
1. تأكد أن كلمة المرور في ملف `.env` صحيحة
2. كلمة المرور الافتراضية: `postgres`
3. إذا غيرتها، عدّل ملف `.env`

### مشكلة: "database 'noble_drip' does not exist"

**الحل:**
1. افتح Terminal
2. اكتب:
   ```
   psql -U postgres
   ```
3. اكتب كلمة المرور
4. اكتب:
   ```sql
   CREATE DATABASE noble_drip;
   \q
   ```

### مشكلة: "port 3000 is already in use"

**الحل:**
1. قد يكون هناك برنامج آخر يستخدم المنفذ 3000
2. أغلق البرامج الأخرى
3. أو استخدم منفذ آخر:
   ```
   npm run dev -- -p 3001
   ```
4. ثم افتح: http://localhost:3001

### مشكلة: الصفحة لا تظهر بشكل صحيح

**الحل:**
1. امسح ذاكرة التخزين المؤقت في المتصفح:
   - `Ctrl + Shift + Delete`
   - اختر "Cached images and files"
   - اضغط "Clear data"
2. أعد تحميل الصفحة: `F5` أو `Ctrl + R`

## 💡 نصائح مهمة

### 1. لا تغلق Terminal
- Terminal حيث يعمل `npm run dev` يجب أن يظل مفتوحاً
- إذا أغلقته، سيتوقف الخادم
- يمكنك فتح Terminal جديد لأوامر أخرى

### 2. احفظ الملفات دائماً
- بعد أي تعديل، اضغط `Ctrl + S`
- VS Code سيحفظ تلقائياً أحياناً، لكن الأفضل حفظ يدوياً

### 3. راقب Terminal
- أي أخطاء ستظهر في Terminal
- اقرأ رسائل الخطأ بعناية
- ابحث عن الحلول على الإنترنت

### 4. استخدم الاختصارات
- `Ctrl + S`: حفظ الملف
- `Ctrl + C`: نسخ
- `Ctrl + V`: لصق
- `Ctrl + Z`: تراجع
- `Ctrl + F`: بحث
- `Ctrl + ~`: فتح/إغلاق Terminal

### 5. كلمات المرور المهمة
